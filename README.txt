# dftbplus

This repository contains files pertaining to DFTB calculations using the program DFTB+.

The directory python contains python scripts for analysis (calculating dipoles) and preparing inputs.
The dftb_inputs directory contains input files and templates for running DFTB+.
The logs directory contains outputs from various script and DFTB+ runs.
The scripts directory contains BAsh and Slurm scripts for preparing, running, and checking calculations.
    The cp2k-traj directory includes calculations using DFTB+ from the cp2k DFT trajectory.
    The ipi-traj directory has files for running DFTB+/i-pi calculations and the associated travis calculations.
The data directory contains pdb and other input files.
